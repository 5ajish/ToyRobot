# bmad-core

BMAD-METHOD (Breakthrough Method of Agile AI-driven Development) core agent framework.

## Setup

Copy the `.bmad-core` folder into your project root, then configure `.bmad-core/core-config.yml` for your project structure.

```bash
# Or install via npm
npx bmad-method install
```

## Usage

See `.bmad-core/data/bmad-kb.md` for the full knowledge base and workflow documentation.
